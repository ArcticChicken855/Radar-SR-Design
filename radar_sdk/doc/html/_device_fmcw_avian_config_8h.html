◆ ifx_Avian_Power_Mode_t

enum ifx_Avian_Power_Mode_t

This enumeration type lists the available power modes of an Avian sensor, that can be configured at the end of a shape or frame.

These power values are used during shape end delay and frame end delay.

Enumerator
IFX_POWER_MODE_ACTIVE 

No power saving.

IFX_POWER_MODE_IDLE 

Go to Idle Mode after last chirp.

IFX_POWER_MODE_DEEP_SLEEP 

Go to Idle Mode after last chirp.

IFX_POWER_MODE_DEEP_SLEEP_CONTINUE 

Go to Deep Sleep Mode after last chirp, but keep system clock enabled.
