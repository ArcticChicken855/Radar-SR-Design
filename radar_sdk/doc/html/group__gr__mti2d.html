Detailed Description

API for 2D Moving Target Indicator filter (2DMTI)

The Moving Target Indicator (MTI) is a radar targeting method that helps discriminating moving targets from static targets and clutter.

The formulae are equivalent to the 1D MTI case. For more information refer to the documentation of Moving Target Indicator (MTI).
