◆ ifx_rai_get_range_doppler()

IFX_DLL_PUBLIC ifx_Cube_C_t* ifx_rai_get_range_doppler ( ifx_RAI_t *  handle)

Getter function to access Range Doppler result.

Returns the pointer to the Range-Doppler map as cube with dimensions range_fft_size (rows) x doppler_fft_size (cols) x num_virtual_antennas (slices).

The ownership remains within the Range Angle Image. The caller must not free the memory of the returned pointer.

Parameters
[in]handleRange Angle Image instance
Returns
Pointer to Range Doppler map
