Detailed Description

API for Pre-processed FFT.

Pre-processed FFT is a 1D FFT along with pre-processing like mean removal and windowing. Following list of signal processing blocks are used within this module:

  1. mean removal (optional)
  2. windowing
  3. FFT (R2C or C2C)
