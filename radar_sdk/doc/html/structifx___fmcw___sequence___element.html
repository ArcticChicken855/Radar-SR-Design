Detailed Description

An element of an acquisition sequence.

This structure is the base of all acquisition sequence elements. An application must never use this type directly, but build acquisition sequences from ifx_Fmcw_Sequence_Loop_t, ifx_Fmcw_Sequence_Chirp_t and ifx_Fmcw_Sequence_Delay_t.

Sequence elements are concatenated to an acquisition sequence as a single linked list.
