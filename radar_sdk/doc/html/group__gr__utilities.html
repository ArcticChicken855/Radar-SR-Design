◆ ifxu_npy_read()

void* ifxu_npy_read ( const char *  filename,
int *  type 
)

Reads .npy file.

Read the .npy filename given by filename. If successful, the pointer to the corresponding ifx data type is returned. The data type is written to type.

The caller must cast the pointer to the correct data type:

  • type=IFX_TYPE_VECTOR_REAL: ifx_Vector_R_t*
  • type=IFX_TYPE_VECTOR_COMPLEX: ifx_Vector_C_t*
  • type=IFX_TYPE_MATRIX_REAL: ifx_Matrix_R_t*
  • type=IFX_TYPE_MATRIX_COMPLEX: ifx_Matrix_C_t*
  • type=IFX_TYPE_CUBE_REAL: ifx_Cube_R_t*
  • type=IFX_TYPE_CUBE_COMPLEX: ifx_Cube_C_t*

If an error occurred NULL is returned and type is set to IFX_TYPE_INVALID.

The caller is responsible to release the memory after use for the vector/matrix/cube.

The npy file format is described in https://numpy.org/neps/nep-0001-npy-format.html.

Parameters
[in]filenamepath to .npy file
[out]typetype of returned object
Return values
pointerto object type if successful
NULLis an error occurred

Write .npy file.

Write raw data to .npy file given by filename. If successful, it will return true.

example queue array parameters for shape matrix:

  • 4D: (frames, antennas, chirps, samples)
  • 3D: (rows, cols, slices)
  • 2D: (rows, cols)
  • 1D: (len)
