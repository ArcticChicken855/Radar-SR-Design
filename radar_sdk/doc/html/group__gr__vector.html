Detailed Description

API for operations on Vector data structures.

Supports mathematical and other operations such as creation and destruction of vectors, or printing vector samples onto a file.

ifx_Complex_t arr_data[4] = {{1, 1}, {2, 2}, {3, 3}, {4, 4}};
