Performs Cholesky decomposition of A.

Compute the Cholesky decomposition of the hermitian and positive definite matrix A

\[ A = L L^\dagger, \]

where L is a complex lower triangular matrix with real and positive diagonal elements. Only the lower triangular part of A (matrix elements on the diagonal and below the diagonal) are read.

If A is not positive definite, IFX_ERROR_MATRIX_NOT_POSITIVE_DEFINITE is set.

Parameters
[in]Amatrix A
[out]Llower triangular matrix

Performs Cholesky decomposition of A.

Compute the Cholesky decomposition of the real symmetric and positive definite matrix A

\[ A = L L^\mathrm{T}, \]

where L is a real lower triangular matrix with positive diagonal elements. Only the lower triangular part of A (matrix elements on the diagonal and below the diagonal) are read.

If A is not positive definite, IFX_ERROR_MATRIX_NOT_POSITIVE_DEFINITE is set.

[in]Amatrix A
[out]Llower triangular matrix

Computes inverse of a generic complex matrix.

This function computes the inverse matrix of A ( \(A^{-1}\)) and saves it in Ainv.

This function works for generic complex square matrices. If A is (numerically) singular, IFX_ERROR_MATRIX_SINGULAR is set.
