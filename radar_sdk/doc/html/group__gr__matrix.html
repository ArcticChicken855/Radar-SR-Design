IFX_DLL_PUBLIC void ifx_mat_copy_c ( const ifx_Matrix_C_t *  from,
ifx_Matrix_C_t *  to 
)

Copies the elements from a complex source matrix to a complex destination matrix.

The condition is that both source and destination matrix instance should be allocated and populated (pointers inside the matrix instance should not be null) using the ifx_mat_create_c. No memory allocation is done in this function.
