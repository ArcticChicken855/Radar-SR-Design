◆ ifx_cw_capture_frame()

IFX_DLL_PUBLIC ifx_Matrix_R_t* ifx_cw_capture_frame ( ifx_Device_Cw_t *  handle,
ifx_Matrix_R_t *  frame 
)

This method captures one frame of raw data.

This function captures for all activated antennas samples_per_antenna and returns the samples as a matrix with dimensions num_antennas (rows) x samples_per_antenna (columns).

The samples of the raw data are normalized to values between -1...1. To convert the samples to voltages, each sample must be multiplied by the reference voltage 1.21V.

If frame is NULL, memory for the matrix will be allocated and returned. Otherwise the memory of frame will be used.

Parameters
[in]handleA handle to the CW device
[in]framePointer to the ifx_Matrix_R_t frame where raw data is stored. If this is NULL, then a new frame is created. the caller is responsible to deallocate the memory.
Returns
pointer to the ifx_Matrix_R_t frame containing the received samples.

◆ ifx_cw_create()

IFX_DLL_PUBLIC ifx_Device_Cw_t* ifx_cw_create ( )

This function creates and initializes the continuous wave mode device instance.

The provided port within the device handle is used to detect the sensor type of the connected device. According to the detected sensor type the CW mode device is initialized. If the detected device is not recognized, an error is set.

After initialization the connection is still in deep sleep mode.

Returns
A handle to the CW device

◆ ifx_cw_create_by_uuid()

IFX_DLL_PUBLIC ifx_Device_Cw_t* ifx_cw_create_by_uuid ( const char *  uuid)

Creates a device handle.

This function opens the CW radar sensor connected to the host machine with the unique id given by UUID.

For converting the UUID from byte array to/from string see

See also
ifx_uuid_from_string and
ifx_uuid_to_string.

On failure one of the following error codes are set:

  • IFX_ERROR_ARGUMENT_INVALID : The format of the UUID is wrong.
  • IFX_ERROR_NO_DEVICE : Sensor with given UUID not found.
  • IFX_ERROR_MEMORY_ALLOCATION_FAILED : Memory allocation failed.
  • IFX_ERROR_NOT_SUPPORTED : Sensor is not an FMCW sensor.
  • IFX_ERROR_FIRMWARE_VERSION_NOT_SUPPORTED : Firmware version not supported anymore.
  • IFX_ERROR_COMMUNICATION_ERROR : A communication error occurred.
  • IFX_ERROR : An unknown error occurred.
Parameters
[in]uuidThe unique ID (UUID) as a string in the format xxxxxxxx-xxxx-Mxxx-Nxxx-xxxxxxxxxxxx
Returns
Handle to the newly created instance or NULL in case of failure.

◆ ifx_cw_get_list()

IFX_DLL_PUBLIC ifx_List_t* ifx_cw_get_list ( void  )

This function returns a list of available devices.

The function returns a list of available CW radar devices. Each list element is of type ifx_Radar_Sensor_List_Entry_t.

The list must be freed after use using ifx_list_destroy.

Here is an example how to use the function:

ifx_List_t* device_list = ifx_fmcw_get_list();
for (size_t i = 0; i < ifx_list_size(device_list); i++)
{
ifx_Radar_Sensor_List_Entry_t* entry = ifx_list_get(device_list, i);
// ...
}
ifx_list_destroy(device_list);

◆ ifx_cw_load_register_file()

IFX_DLL_PUBLIC void ifx_cw_load_register_file ( ifx_Device_Cw_t *  handle,
const char *  filename 
)

Load register list to a file.

This function loads a register list from a file and applies the CW device configuration represented by that file.

Warning
There is no plausibility check on the loaded registers. A bad register set list may not work at all or load to unexpected behavior. Especially register lists made for a different device type usually don't work.
Parameters
[in]handleA handle to the radar device object.
[in]filenameThe path to the file the register list shall be loaded from.

uint32_t  antenna 
)

This method measures returns the power that is emitted from the specified TX antenna.

The power measurement is only performed if the continuous wave is enabled.

Parameters
[in]handleA handle to the CW device
[in]antennaThe index of the antenna to be enabled or disabled The value is 0 based and must be less than the total number of physical TX antennas within device. If the value is not in the allowed range, an error is set
Returns
If the continuous wave is enabled the measured TX power in dBm is returned. Otherwise -infinity is returned.

IFX_DLL_PUBLIC void ifx_cw_set_adc_config ( ifx_Device_Cw_t *  handle,
const ifx_Cw_Adc_Config_t *  config 
)

This method configures the device's ADC parameters, defined by ifx_Cw_Adc_Config_t.

It includes the configuration of Sample Time ifx_Avian_ADC_SampleTime, tracking or number of conversion ifx_Avian_ADC_Tracking, oversampling factor ifx_Avian_ADC_Oversampling and enable/disable of double the MSB time.

It's recommended to read back the applied value using ifx_cw_get_adc_config.

Parameters
[in]handleA handle to the CW device
[in]configADC configurations defined by ifx_Cw_Adc_Config_t
