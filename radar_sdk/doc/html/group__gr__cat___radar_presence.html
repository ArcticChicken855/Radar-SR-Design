Performs the presence sensing algorithm on given raw data.

Perform the presence sensing algorithm on the given matrix consisting of raw frame data coming from the configured radar transceiver.

The matrix frame_Data consists of frame data from the radar transceiver. The number of rows of the input matrix should match the number of chirps per frame and the number of columns should match the number of samples per chirp expected by the ifx_Presence_Sensing_t.

Parameters
[in]handleHandle to the presence sensing object.
[in]frame_dataCube consisting of raw frame data from the radar transceiver.
[out]resultStructure containing the result of the presence algorithm.
