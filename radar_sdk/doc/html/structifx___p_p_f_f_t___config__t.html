Detailed Description

Defines the structure for pre-processed FFT related settings.

Field Documentation

◆ fft_size

uint32_t fft_size

FFT size must be power of 2 else returns error.

◆ fft_type

ifx_FFT_Type_t fft_type

FFT type is defined by ifx_FFT_Type_t.

Type is defined based on FFT input data nature, i.e. real or complex.

◆ is_normalized_window

bool is_normalized_window

Setting to zero will turn off this feature.

If set to non-zero value the effect of different window type is normalized. Changing window type will result in similar magnitude range.

◆ mean_removal_enabled

bool mean_removal_enabled

Zero value considered as False (skip mean removal feature), while non-zero value would be considered as true for mean removal.
